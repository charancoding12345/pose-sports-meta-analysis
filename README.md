# Meta-Analysis of Pose Estimation Accuracy in Sports and Exercise

This repository contains materials, data, and analysis scripts for a meta-analysis evaluating the accuracy of computer vision–based motion-tracking and pose estimation algorithms in sports and exercise contexts.

## Structure
- `/data/` — raw and processed extraction CSVs  
- `/analysis/` — JASP/Excel meta-analysis files and figures  
- `/docs/` — protocol, PRISMA flow diagram, and supporting materials  
- `/manuscript/` — draft versions and submission files
